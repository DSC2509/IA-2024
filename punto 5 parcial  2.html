segundo parcial pregunta 5

El código logic es una implementación de un sistema para representar y evaluar expresiones lógicas. contine diferemtes partes clave o fundamentales que dentro del algoritmo que son

Clase / Función Descripción Aplicaciones
Sentence Clase base para representar oraciones lógicas. Define métodos abstractos como evaluate, formula y symbols. Utilizada en sistemas de lógica matemática y razonamiento en IA.
Symbol Representa un símbolo lógico (ej. proposiciones como p, q). Implementa métodos para evaluar su valor en un modelo. Permite la representación de proposiciones en un sistema lógico.
Not Representa la negación lógica. Evalúa el valor del operando invirtiéndolo. Usada para construir expresiones complejas a partir de proposiciones simples.
And Representa la conjunción lógica (Y). Evalúa si todas las conjunciones son verdaderas. Permite crear expresiones que deben cumplirse simultáneamente.
Or Representa la disyunción lógica (O). Evalúa si al menos una de las disyunciones es verdadera. Utilizada para expresar opciones o alternativas en proposiciones.
Implication Representa la implicación lógica (si... entonces...). Evalúa la relación condicional entre dos proposiciones. Usada para representar reglas y condiciones en sistemas de conocimiento.
Biconditional Representa la bicondicional lógica (si y solo si). Evalúa si ambas proposiciones tienen el mismo valor de verdad. Permite expresar equivalencias entre proposiciones.
model_check Verifica si una base de conocimiento implica una consulta. Utiliza búsqueda por modelos. Usada en sistemas de razonamiento para validar deducciones lógicas.
